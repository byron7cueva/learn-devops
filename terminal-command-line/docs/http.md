# HTTP

## Comandos HTTP

```bash
# curl Se utiliza para hacer pedidos crudos (raw)
usuario@pc:~$ curl pagina
# Para ver toda la comunicacion http
usuario@pc:~$ curl -v pagina
# Ver encabezados
# /dev/null Es un archivo agujero negro, un archivo que no es un archivo.
usuario@pc:~$ curl -v pagina > /dev/null

# wget Realiza descargas de servidores HTTP
usuario@pc:~$ wget pagina_del_archivo
```
